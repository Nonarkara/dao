# Fork Guide

This project is intentionally forkable. A fork can be:

- a classroom edition
- a Thai-first edition
- a Chinese-study edition
- a Buddhist comparison edition
- a no-comics public-domain-only edition
- a research-heavy edition with footnotes everywhere
- a tiny phone-first edition for reading one chapter per morning

## Easy Changes

- Edit chapter summaries in `summaries.js`.
- Edit the main chapter text in `chapters.js`.
- Edit deeper commentary in `extended.js`.
- Edit religious/psychological parallels in `comparative_notes.js`.
- Edit the reference shelf in `reference_library.js`.

## Medium Changes

- Add a new language toggle.
- Add audio files and wire them into the chapter shell.
- Add a new note overlay.
- Add a print stylesheet.

## Hard Changes

- Split the site into reusable components.
- Create a search index.
- Add offline support.
- Build a guided reading path that remembers where the reader got stuck.

## Good Pull Requests

A good PR changes one idea at a time. It explains the reading problem it solves, includes screenshots if visual, and checks all three language modes.
