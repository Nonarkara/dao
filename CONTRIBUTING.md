# Contributing

Thank you for wanting to make this stranger, clearer, kinder, or more accurate.

This repo welcomes:

- better translations
- better citations
- Buddhist, Daoist, Confucian, Hindu, Christian, Islamic, Stoic, or psychological parallels
- bug fixes
- accessibility fixes
- calmer CSS
- teaching materials
- new language layers

## How To Work

1. Fork the repo.
2. Run it locally with `python3 -m http.server 4177`.
3. Make a focused change.
4. Test the page in English, Thai, and Chinese modes.
5. Open a pull request explaining what changed and why.

## Translation Rule

When a page is in English, Thai, or Chinese mode, the user should not be forced to read a different language for interface text. Exceptions are source text, Chinese chapter titles, pinyin, quoted book titles, and comic panels that are intentionally preserved as images.

When showing an English Tsai comic panel, add a short Thai and Chinese explanation nearby.

## Content Tone

Keep it readable. The project can be scholarly without becoming stiff. Good notes should help a tired reader understand the chapter, not prove that the note-writer owns a library card.

## Attribution

Please credit translators, scholars, artists, and source books. If you add third-party images, say where they came from and why they are included.
